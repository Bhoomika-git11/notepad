# Notepad++ Clone — Python Tkinter

A lightweight Notepad++ inspired editor built with pure Python. No external libraries needed.

## Features
- Multi-tab editing with close buttons
- Python syntax highlighting
- Line numbers & status bar
- Find & Replace (with regex)
- Toolbar with tooltips & right-click menu
- Font picker, Zoom, Word Wrap

## Run
```bash
python notepad_plus_plus.py
```
> Requires Python 3.10+

## Shortcuts
| Shortcut | Action |
|----------|--------|
| `Ctrl+N/O/S` | New / Open / Save |
| `Ctrl+Z/Y` | Undo / Redo |
| `Ctrl+X/C/V` | Cut / Copy / Paste |
| `Ctrl+F` | Find & Replace |
| `Ctrl+W` | Close tab |
| `Ctrl+±` | Zoom In / Out |
